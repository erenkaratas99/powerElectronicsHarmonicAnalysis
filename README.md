# powerElectronicsHarmonicAnalysis
It analysis the parameters of the inverted AC harmonics at the output of the DC 2 AC inverter

Inputs : 
  -n (harmonic amount)
  -frequency (Hz)
  -DC V (DC Component)
  -Load params (R : ohms, L : Henry)
  etc.
  
 Output : 
   -Impedances
   -Currents
   -Voltages
   -Consumed power
   -THD (Total Harmonic Distortion)
  of the harmonics
